Under The Hood

Technology & Security

Built with industry-standard cryptography and secure key derivation

Key Derivation

MassaConnect uses hierarchical deterministic (HD) wallets following BIP-32/BIP-39/BIP-44 standards for key generation and management.

  • 24-word mnemonic phrase (BIP-39)
  • HD derivation path: m/44'/632'/0'/0'
  • Coin type 632 for Massa blockchain
  • Hardened derivation for enhanced security

Elliptic Curve

Massa blockchain uses the secp256k1 elliptic curve, the same curve used by Bitcoin and Ethereum, ensuring battle-tested cryptography.

  • secp256k1 curve (256-bit)
  • ECDSA signatures
  • Keccak-256 for address derivation
  • Base58Check address encoding

Local Security

All sensitive data is encrypted and stored locally on your device using Android's security framework.

  • Android Keystore for key storage
  • AES-256 encryption at rest
  • Biometric authentication support
  • No cloud sync or remote storage

Network & RPC

Direct communication with Massa blockchain nodes via JSON-RPC for all operations.

  • Official Massa mainnet/buildnet nodes
  • Configurable custom RPC endpoints
  • Automatic retry with failover
  • No third-party API dependencies

✓ Compatible With

  • Massa Station wallet (import/export)
  • Bearby browser extension
  • Any wallet using BIP-39 mnemonics
  • Massa blockchain mainnet & buildnet
  • MRC-20 tokens (fungible)
  • MRC-721 tokens (NFTs)

âœ— Not Compatible With

  • Ethereum/EVM wallets (different address format)
  • Ledger hardware wallets (no Massa app yet)
  • WalletConnect protocol (not supported on Massa)
  • Non-standard derivation paths
Help Center

Frequently Asked Questions

Find answers to common questions about MassaConnect

What is MassaConnectâ€¢
MassaConnect is a self-custodial mobile wallet for the Massa blockchain. It allows you to securely store, send, receive, and swap MAS tokens and other assets on the Massa network. Your private keys are stored locally on your device and never shared with anyone.
Is MassaConnect safe to useâ€¢
Yes, MassaConnect is designed with security as a top priority. It's 100% open source (you can verify the code), uses industry-standard cryptography (secp256k1, AES-256), and stores all keys locally using Android Keystore. The app has been verified clean by VirusTotal (0/71 detections) and collects zero user data.
Can I import my Massa Station walletâ€¢
Yes! MassaConnect is fully compatible with Massa Station and Bearby wallets. You can import your existing wallet using your 24-word recovery phrase. The derivation path m/44'/632'/0'/0' is the same across all Massa wallets.
What tokens are supportedâ€¢
MassaConnect supports MAS (native token), all MRC-20 fungible tokens, and MRC-721 NFTs on the Massa blockchain. You can add custom tokens by entering the contract address, and the app will automatically fetch token details.
How does staking workâ€¢
You can stake MAS by purchasing "rolls". Each roll costs 100 MAS. When you own rolls, you participate in block production and earn rewards. MassaConnect shows real-time APY estimates and allows you to buy/sell rolls directly from the app.
Is my recovery phrase stored securelyâ€¢
Your recovery phrase is encrypted using AES-256 and stored in Android's secure enclave (Keystore). It never leaves your device and is never transmitted over the network. Make sure to write down your phrase and store it safely offline.
Why can't I see my NFTsâ€¢
MassaConnect scans known NFT collections automatically. If your NFT is from a newer or unknown collection, you can manually add it using the "Add Collection" feature. Enter the smart contract address and the app will discover your NFTs.
What's the derivation path usedâ€¢
MassaConnect uses the standard Massa derivation path: m/44'/632'/0'/0' where 632 is the registered coin type for Massa. This ensures compatibility with other Massa wallets like Massa Station and Bearby.
What if MassaConnect stops working or gets removedâ€¢
Your funds are always safe! MassaConnect is just a window to the blockchain - your crypto lives on Massa, not in the app. If MassaConnect ever stops working, simply import your 24-word recovery phrase into Bearby (browser extension) or Massa Station (desktop wallet). We use the same derivation path m/44'/632'/0'/0', so your seed will generate the exact same address. This is the power of self-custody: you're never locked into one app.
Why does the dApps browser use Bearby instead of MassaConnectâ€¢
When you connect to dApps through MassaConnect's built-in browser, the connection uses the Bearby protocol. This is because most dApp frontends (like Dusa, NFT marketplaces, etc.) only have Bearby integration available as their wallet connection option - they haven't added MassaConnect as a native choice yet.

Don't worry - it's still your wallet! MassaConnect injects Bearby-compatible signatures so you can use any Massa dApp seamlessly. Your transactions are signed with your MassaConnect private keys, not Bearby's. As more dApps add native MassaConnect support, we'll update the integration.
